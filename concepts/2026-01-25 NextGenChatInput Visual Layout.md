## Responsive Behavior

### Desktop (> 768px)
- Full width with max-w-4xl container
- Model selector always visible (if allowed)
- All controls in single row

### Mobile (< 768px)
- Full width input
- Model selector stacks or shows abbreviated name
- Touch-friendly button sizes (p-2.5)

## Color States

### Send/Stop Button States
1. **Disabled** (no input): `bg-gray-200 text-gray-400`
2. **Ready** (with input): `bg-indigo-500 text-white hover:bg-indigo-600`
3. **Processing**: `bg-red-500 text-white hover:bg-red-600`

### Model Selector States
1. **Closed**: `bg-gray-100 text-gray-700 hover:bg-gray-200`
2. **Open**: Same style with dropdown visible
3. **Selected item**: `bg-indigo-50 text-indigo-600`

### Input Focus
- Border: `focus-within:ring-2 focus-within:ring-indigo-500`
- Highlight: Indigo ring around entire component

## Dark Mode

All components support dark mode variants:
- Background: `dark:bg-gray-800`
- Border: `dark:border-gray-600`
- Text: `dark:text-gray-100`
- Hover states: `dark:hover:bg-gray-700`

## Keyboard Shortcuts

| Key Combination | Action |
|----------------|--------|
| Enter | Send (single-line mode) |
| Shift+Enter | New line (multiline mode) |
| Cmd+Enter / Ctrl+Enter | Send (any mode) |
| / (empty input) | Open prompt search |
| Esc | Close dropdowns/menus |
