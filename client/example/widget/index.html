AI Hub Apps Chat Widget Demo

This page demonstrates how to integrate the AI Hub Apps Chat Widget into your own website.

Try It Out

Click the button in the bottom right corner to open the chat widget. You can interact with our AI assistant directly from this page.

Installation

To add the AI Hub Apps Chat Widget to your website, include our script and initialize it with your configuration:

<!-- Add this to your HTML head or body -->
<script src="https://your-domain.com/ai-hub-widget.js"></script>
<script>
  // Initialize the widget once the page is loaded
  document.addEventListener('DOMContentLoaded', function() {
    aiHubWidget.init({
      // Optional: Specify which app to use
      appId: 'general-assistant',
      
      // Optional: Custom branding color
      primaryColor: 'rgb(0, 53, 87)',
      
      // Optional: Widget position ('right' or 'left')
      position: 'right',
      
      // Optional: Language ('en', 'de', etc.)
      language: 'en'
    });
  });
</script>

Customization

You can customize the appearance and behavior of the widget through the initialization options:

  • appId: Specify which AI app to use (defaults to the system default)
  • serverUrl: Set a custom server URL (defaults to the current domain)
  • position: Position of the widget ('right' or 'left')
  • primaryColor: The main color for the widget
  • language: The language to use ('en', 'de', etc.)
