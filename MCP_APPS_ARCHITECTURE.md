## Key Technologies

- **Frontend**: React, Hooks, postMessage API
- **Backend**: Express.js, JSON-RPC 2.0, Zod validation
- **Security**: iframe sandbox, CSP, rate limiting
- **Protocol**: JSON-RPC 2.0 over postMessage
- **Storage**: File-based (HTML/JS/CSS)
